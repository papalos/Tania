=== Работа с программой "Tania - meow sender" ===

--------------- Подготовка данных ---------------

1) Создайте CSV файл (разделитель ";") со следующей последовательностью колонок:
   address - адреса получателя письма
   subject - тема письма
   body - содержание письма, разметка выполняется html тегами.
   attach - путь до файла вложения
   -- формирования данного файла можно выполнить в Excel выполнив автоподсканулку фио,
      и имени соотносимого файла персонального вложение,
      после чего выполнить сохрание полученной таблице в файл формата CSV (разделитель - запятая)

------------ Работа с программой ----------------
2) Проверьте наличие файла настройки (setup.csv) в директроии с программой.
3) Запустите программу
4) Проврьте настройки выполнив нажатие на кнопку "Настройки",
   и, по необходимости, внесите изменения, подтвердив их нажатием кнопки "Сохранить"
   -- по умолчанию используется: server:smtp.hse.ru, port:587, email:olymp@hse.ru, login:..., password:...
5) Закройте окно настроек
6) Выберите файл с данными об адресатах рассылки
7) Нажмите на кнопку "Отправить рассылку"
8) По завершении рассылки будет выведено сообщение.
